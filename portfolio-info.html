About This Portfolio

Design decisions and technical details

Portfolio Description

This portfolio website was designed to showcase my skills, projects, and professional journey as a software developer. The primary goals were:

  • Demonstrate my technical skills through a well-structured, responsive website
  • Showcase my projects with detailed descriptions and links to code repositories
  • Provide potential employers or collaborators with a way to contact me
  • Reflect my personal brand as a developer

The portfolio was built using Bootstrap 5 for responsive design and layout, with custom CSS for additional styling. Animations were implemented using Animate.css for subtle interactive elements.

Design Inspirations

Minimalist Design

Inspired by clean, minimalist portfolios that focus on content rather than excessive decoration.

Dark Theme

Many developer portfolios use dark themes which are easier on the eyes for prolonged viewing.

Interactive Elements

Subtle animations and hover effects to make the experience more engaging without being distracting.

Color Scheme

Primary (#0d6efd)

Secondary (#6c757d)

Dark (#212529)

Light (#f8f9fa)

The color scheme uses Bootstrap's default dark theme with primary blue accents for a professional, developer-friendly appearance that's easy on the eyes.
